  # 易經占卜 - 用戶支援

  ## 📱 App 資訊

  **易經占卜：占卦與學習** 是一款現代化的易經占卜與學習應用程式。

  - **平台：** iPhone Only（iOS 17.0+）
  - **語言：** 繁體中文 / English / 日本語
  - **版本：** v1.0

  ## ✨ 主要功能

  - **梅花易數**：時間取數或手動輸入三數成卦
  - **金錢卦**：傳統三枚硬幣占卜，支援閃電模式
  - **64卦資料庫**：完整的學習系統
  - **歷史記錄**：本地安全儲存
  - **三語系支援**：即時語言切換

  ## 💬 常見問題

  ### Q: App 會收集我的個人資料嗎？
  A: 不會。所有資料（包括您的占卜問題）僅儲存在您的裝置上，不會上傳到任何伺服器。

  ### Q: 可以備份占卜記錄嗎？
  A: 目前記錄僅存於本機。我們正在考慮未來版本加入 iCloud 同步功能。

  ### Q: 支援 iPad 嗎？
  A: 目前僅支援 iPhone。iPad 支援正在規劃中。

  ### Q: 如何切換語言？
  A: 進入 App 的「資訊」頁面，選擇您偏好的語言即可即時切換。
